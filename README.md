## 🚀 How to Run

1. Extract the ZIP file
2. Open `index.html` in any modern browser
3. No installation, no build tools, no server needed!

For live hosting:
- Upload all 3 files to **Netlify**, **GitHub Pages**, or any shared hosting
- Keep all files in the **same folder**
